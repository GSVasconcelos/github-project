### Project to complete the GitHub project challenge by DIO: learn how to use the GitHub.

Este projeto foi criado para cumprir um requisito do curso de GitHub da DIO, que visa ensinar aos desenvolvedores os fundamentos do Git e GitHub, criar repositórios remotos e vinculá-los aos seus projetos, fazer mudanças de código locais usando _commits_ e enviando pros repositórios remotos, etc.

## Tecnologias: 
- Git
- GitHub
- Markdown

## Códigos (terminal):
- `commit`
- `branch`
- `merge`
- `checkout`
- `clone`
- `init`
- `status`
- `add .`
- `pull`
- `push`
- `fetch`

## Objetivos: 
[X] Criar um repositório e vinculá-lo a um projeto; <br>
[X] Criar uma _branch_, fazer um _commit_, efetuar um _merge_ e dar _push_; <br>
[] Adicionar um colaborador ao projeto, fazer um fork e ambos contribuir nele; <br>
[] Fazer um README.me do projeto e aplicar as técnicas ensinadas. <br>
